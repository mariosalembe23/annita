# Política de Segurança

## Versões Suportadas

Atualmente mantemos atualizações de segurança apenas para a versão mais recente da Annita.

| Versão | Suportada |
|---|---|
| latest (`main`) | ✅ |
| versões anteriores | ❌ |

---

## Reportar uma Vulnerabilidade

**Por favor, não reportes vulnerabilidades de segurança através de issues públicas.**

Se descobriste uma vulnerabilidade de segurança na Annita, agradecemos que a reportes de forma responsável:

### Como reportar

Envia um email para **[geral.annita@gmail.com](mailto:geral.annita@gmail.com)** com o assunto:

```
[SECURITY] Descrição breve da vulnerabilidade
```

Inclui no email:
- Descrição detalhada da vulnerabilidade
- Passos para reproduzir o problema
- Impacto potencial
- Sugestão de correção (se tiveres)

### O que esperar

- **Confirmação de receção**: dentro de 48 horas
- **Avaliação inicial**: dentro de 5 dias úteis
- **Resolução e disclosure**: coordenada contigo antes de ser tornada pública

Agradecemos a tua colaboração em manter a Annita segura para toda a comunidade. 🙏
